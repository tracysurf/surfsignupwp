== Description ==

DW Knowledge Base is a free WordPress plugin that allows users to create a self-support customer services (a knowledgebase system). This Knowledge Base plugin focuses on user's experience with useful features such as integrating with almost WordPress themes, but powerful with a auto-complete search form, multiple widgets, breadcrumbs, comments, short-codes and others. Premium version is available with more great features and layouts.

= Key features =
* Quick installation and setup.
* Full responsive layout in all mobile devices.
* Flexible integrating with all popular WordPress themes.
* Widgets available: Categories, Tags, Popular Articles, Random Articles, Latest Articles.
* Display article views on your WordPress website.
* Enable/disable search form.
* Turn on/off comment.
* Turn on/off breadcrumbs.
* Using [dwkb] shortcode to display knowledge base index page.
* Display related posts.

== Frequently Asked Questions ==

= How to fix `Not Found Post` ? =
Go to `Dashboard > Settings > Permalinks`, click on "Save Changes" button to re-update the permalink.

== Installation ==

1. Upload `dw-knowledge-base` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Pages > Add New`, create a new page, then add the short-code [dwkb] to the editor area of the page.
4. Navigate to the Knowledge Base tab that appears in your Admin Menu, go to Settings, select the Index page as the page you have created from step 3.
